Project Description

Ladigue Island Quest Seychelles is a lightweight choose-your-path web application designed as a fun, story-based experience. Users read short scenes and make decisions using clear buttons. The app is built using HTML for structure, CSS for presentation, and JavaScript for game logic and branching.

Why it works this way

The interface is intentionally simple: the story text is the focus, while large buttons make choices easy on mobile and desktop. Restart and Home controls ensure users never get trapped in one path. Scene achievements and saved progress (via browser storage) add replay value without requiring a server.
